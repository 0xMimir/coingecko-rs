# coingecko-rs

Rust library for the CoinGecko V3 API

## Installation

Add the following to your `Cargo.toml` file:

```toml
[dependencies]
coingecko-rs = "0.1.3"
```

## Features

- Supports all API endpoints
- Responses are fully typed using `serde_json`
